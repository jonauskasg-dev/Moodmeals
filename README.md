# MoodMeals

MoodMeals - это landing page для фиктивного веб-приложения. Идея простая: пользователь выбирает, как он себя чувствует сегодня, а страница предлагает блюдо, подходящее под это настроение.

## Использованные технологии

- HTML
- CSS
- JavaScript
- Google Fonts
- фотографии Unsplash

Проект полностью статичный, поэтому ему не нужны backend, база данных, API-ключи, Node.js сервер или build-процесс.

## Как запустить локально

1. Скачай или скопируй файлы проекта.
2. Открой `index.html` в браузере.

Также можно запустить проект через простой статичный сервер, но это не обязательно.

## Как загрузить на GitHub Pages

1. Загрузить файлы в GitHub repository.
2. Перейти в Settings -> Pages.
3. Выбрать branch `main`.
4. Выбрать root folder.
5. Сохранить и дождаться публикации страницы.
